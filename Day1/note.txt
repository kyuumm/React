react是一个构建web和原生交互界面的库

创建react开发环境工具
npx create-react-app react-basic
npx:nodejs工具命令
中间：核心包，用于创建react项目
react-basic:react项目的名称

跑项目：
  cd react-basic
  npm start

  npm build是打包（开发完毕后使用）
